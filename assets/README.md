# Screenshots

Drop your workflow screenshots here with these exact filenames so the README and guide
render them automatically:

| Filename | What to capture |
|---|---|
| `workflow-overview.png` | The whole canvas (all three flows) — used as the hero image |
| `flow-sender.png` | The sender row (Schedule → Decide → Claude → Send → Mark sent) |
| `flow-followups.png` | The follow-up row |
| `flow-replies.png` | The reply-detection row |
| `sheet-example.png` | The Google Sheet with the column headers + a few example rows |

Tips for clean screenshots:
- Zoom the n8n canvas so node labels are readable, then use a window/area screen capture.
- PNG, ideally 1600px+ wide.
- Hide any real lead data / your email in the sheet screenshot (blur or use dummy rows).
